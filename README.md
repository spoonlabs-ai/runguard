# RunGuard

Runtime safety for agents: watchdog timeouts, budget controls, circuit breakers, structured traces.

## Install

```bash
npm install
npm link
```

## CLI

```bash
runguard --help
runguard exec -- node -v
```

Gate A: CLI skeleton + locked config schema. Safety enforcement comes in follow-up issues.

- Config schema: `docs/config-schema-v0.1.md`
